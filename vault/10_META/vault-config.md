# Vault Configuration

This vault is built on the **Creator System** by [Matty Mo Studio](https://themostfamousartist.com).

## Network

The Creator Network is an opt-in system that connects vaults for cross-referencing and discovery. When enabled, your vault's *structure* (page titles, link graph, themes) can be queried by other network members. Your *content* is never shared without explicit permission.

To opt in, set `opted_into_network: true` in the frontmatter above.

## Version

Update notifications and new skills are distributed through the Creator System. Your current version is shown in the frontmatter above.
