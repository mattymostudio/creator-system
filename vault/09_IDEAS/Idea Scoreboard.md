# Idea Scoreboard

Not every idea deserves oxygen. This is where you decide what earns it.

Score each idea 1–5 on six axes. Be honest — the math punishes wishful thinking.

---

## The six axes

| Axis | The question | 5 = | 1 = |
|---|---|---|---|
| **Heat** | How obsessed are you? | Can't stop thinking about it | Feels like homework |
| **Edge** | How weird, rare, or counter-positioned? | Nobody else is doing this | Everyone already is |
| **Legend** | Does this add to your story? | Chapter-worthy | Footnote at best |
| **Ship** | How fast can this exist in the world? | Days | Years |
| **Yield** | What's the payoff — money, audience, proof, leverage? | Multiple wins stacked | Marginal |
| **Drag** | If it flops, what's the cost? | Devastating | Shrug-worthy |

> Edge gets double weight in the formula. The world is full of 3s on every axis. What's rare is the move that makes people say "wait, what?"

## The formula

```
score = heat + (2 × edge) + legend + ship + yield - drag
```

**Range:** 1 (kill it) → 29 (do it yesterday).

## Stages

| Stage | Score | Translation |
|---|---|---|
| **Now** | 22+ | Start this week |
| **Queue** | 17–21 | Next sprint |
| **Incubate** | 12–16 | Develop the seed |
| **Back burner** | 7–11 | Revisit quarterly |
| **Reframe** | Below 7 | The idea isn't wrong — the shape is |

## How to use this

1. Drop any new idea into [[Idea Template]] first — name it, describe it in two sentences, note why now.
2. Score it here. Don't agonize — first instinct is usually right.
3. Anything that lands in **Now** or **Queue** moves to [[Execution Queue]].
4. Re-score quarterly. Scores drift. A 7 last quarter might be a 22 now.
