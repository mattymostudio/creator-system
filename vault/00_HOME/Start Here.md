# Start Here

This is your Personal Intelligence System.

It is not a notes app. It is a layered knowledge system — for archiving source material, compiling structured knowledge, tracking active work, and generating reusable outputs.

---

## The five jobs of this vault

1. Archive source material
2. Compile raw material into structured knowledge
3. Maintain a living model of life / work / thought
4. Generate reusable outputs
5. Train future agents on this world model

---

## How to navigate

| Layer | Folder | What lives here |
|-------|--------|-----------------|
| Raw sources | `02_SOURCES` | Articles, transcripts, journals, meetings, business docs |
| Processing | `03_SOURCE_NOTES` | Structured notes derived from sources |
| Knowledge | `04_CANON` | People, places, themes, timeline, decisions |
| Execution | `05_PROJECTS` | Active and historical projects |
| Deliverables | `06_OUTPUTS` | Memos, bios, decks, press materials, copy |
| Ideas | `09_IDEAS` | Concepts not yet promoted |
| Inbox | `01_INBOX` | Unprocessed material — transient |
| System | `10_META` | AGENTS.md, templates, schema, workflows |
